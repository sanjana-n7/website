Accessibility & print: - Skip link, focus outlines, aria attributes included. - Print styles configured for A4 portrait. Use "Download as PDF" on the Resume page to open print dialog. Deploy: - Vercel: Connect the repo and deploy (auto-detect static site). No build step required. - Netlify: Drag & drop the folder in Netlify Drop or connect the repo and set publish directory to the repo root. Customization: - Replace all placeholders with your real data. - For production PDF generation with consistent layout consider server-side PDF generation (Puppeteer, wkhtmltopdf) or a client library (jsPDF / html2pdf) — note these may need extra configuration.
